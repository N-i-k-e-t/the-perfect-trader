# THE PERFECT TRADER

Psychology-first trading discipline — rules, mood, journal, and behavioral coaching.

**GitHub:** [the-perfect-trader](https://github.com/N-i-k-e-t/the-perfect-trader)  
**Live (after Vercel import):** connect repo at [vercel.com/new](https://vercel.com/new) — see [docs/guides/LAUNCH.md](docs/guides/LAUNCH.md)

## Quick start (local)

```powershell
npm install
npm run dev
```
